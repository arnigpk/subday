

# Подсказка "Откуда взять ID" на странице приглашения друга

## Что изменится

Под кнопкой "Выдать гостевой доступ" появится информационный блок-подсказка, объясняющий где найти ID друга. Блок будет содержать иконку и пошаговую инструкцию.

## Визуальный вид

```text
+----------------------------------+
|  [поле ввода ID]                 |
|  [Выдать гостевой доступ]        |
|                                  |
|  (i) Где найти ID друга?         |
|  Попросите друга открыть          |
|  приложение -> Профиль.          |
|  ID отображается под аватаром.   |
+----------------------------------+
```

## Технические детали

### Файл: `src/pages/GiftCoffeePage.tsx`

Добавить после кнопки "Выдать гостевой доступ" информационный блок:
- Использовать иконку `Info` из `lucide-react`
- Стилизовать как мягкий блок с `bg-muted/50 rounded-xl p-4`
- Текст: заголовок "Где найти ID друга?" и описание: "Попросите друга открыть приложение, перейти в раздел Профиль — ID отображается под аватаром"
- Добавить переводы в `useLanguage` для казахского языка через ключи `guest.whereToFindId` и `guest.whereToFindIdDesc`

### Файл: `src/contexts/LanguageContext.tsx`

Добавить ключи переводов:
- `guest.whereToFindId`: "Где найти ID друга?" / "Достың ID-ін қайдан табуға болады?"
- `guest.whereToFindIdDesc`: "Попросите друга открыть приложение и перейти в раздел «Профиль». ID отображается под аватаром." / казахский перевод
