

# Казахский орнамент в шапке главной страницы

## Подход

Сгенерировать казахский орнамент через встроенную AI-модель генерации изображений (google/gemini-2.5-flash-image), сохранить его в storage, и применить как фоновый паттерн шапки на главной странице.

## Шаги реализации

1. **Генерация орнамента** — через Edge Function вызвать AI-модель для создания бесшовного казахского орнамента (золотисто-карамельные тона, прозрачный/полупрозрачный фон, горизонтальная полоса)

2. **Сохранение изображения** — загрузить результат в файл проекта (например `src/assets/kz-ornament.png`) или в storage bucket

3. **Применение в шапке** — обновить header в `src/pages/HomePage.tsx`:
   - Добавить фоновое изображение с `background-repeat: repeat-x`
   - Полупрозрачность орнамента чтобы текст оставался читаемым
   - Адаптивность на всех размерах экрана

## Технические детали

### Файлы для изменения
- `src/pages/HomePage.tsx` — добавить фоновый слой с орнаментом в блок header
- Новый файл: `src/assets/kz-ornament.png` — сгенерированное изображение

### Стилизация шапки
```text
Header container
+-- Ornament background layer (absolute, opacity 0.15-0.25, repeat-x)
+-- Content (greeting, logo, admin button) — поверх орнамента
```

### Параметры генерации
- Стиль: традиционный казахский орнамент (бараний рог / кошкар муйиз)
- Цвета: золотой/карамельный на прозрачном фоне
- Размер: ~1200x200px, бесшовный по горизонтали
- Прозрачность при наложении: 15-25% чтобы не мешать чтению
