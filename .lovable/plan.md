

## Вариант D: Иконка увеличения + одноразовый тултип

### Что делаем

1. **Постоянная иконка `Maximize2` в правом нижнем углу каждой картинки**
   - Полупрозрачная, небольшая (16px), на подложке `bg-black/30 backdrop-blur-sm`
   - Кликабельна — открывает lightbox (тот же onClick что и у картинки)

2. **Одноразовый тултип-подсказка** при первом появлении поста с картинкой
   - Текст: «Нажмите на фото для полноэкранного просмотра» (с переводом через `t()`)
   - Появляется над иконкой, исчезает через 3 секунды или по тапу
   - Флаг `subflow_image_hint_shown` сохраняется в `localStorage` — показываем только один раз

### Файлы

- **`src/components/subflow/SubFlowPost.tsx`** — добавить иконку поверх картинки (absolute позиционирование внутри контейнера изображения), логику тултипа с `localStorage` проверкой
- **`src/contexts/LanguageContext.tsx`** (или файл переводов) — добавить ключ перевода для подсказки

### Техническая реализация

- Иконка рендерится внутри `relative`-контейнера картинки, позиция `absolute bottom-2 right-2`
- Тултип — простой `div` с `animate-fade-in` + `setTimeout` для автоскрытия через 3 сек
- При первом рендере поста с картинкой проверяем `localStorage.getItem('subflow_image_hint_shown')` — если нет, показываем тултип и ставим флаг
