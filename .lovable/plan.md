

# AI-ассистент для пользователей subday

## Что будет сделано

Добавим плавающую кнопку-помощника на всех страницах приложения. При нажатии откроется чат с AI-ассистентом, который знает всё о приложении subday и помогает пользователям разобраться с функциями.

## Как это будет работать

1. На каждой странице в левом нижнем углу (над нижней навигацией) появится кнопка с иконкой чата
2. При нажатии открывается Sheet (выезжающая панель снизу) с чатом
3. Пользователь пишет вопрос -- AI отвечает в режиме стриминга (токен за токеном)
4. AI знает структуру приложения: подписки, кофейни, бонусы, стрики, QR-коды, профиль и т.д.

## Архитектура

```text
+------------------+       +---------------------+       +-------------------+
|  Кнопка-чат      | ----> |  Sheet с чатом      | ----> |  Edge Function    |
|  (на всех стр.)  |       |  (стриминг ответов) |       |  ai-assistant     |
+------------------+       +---------------------+       +-------------------+
                                                                  |
                                                          +-------------------+
                                                          |  Lovable AI       |
                                                          |  Gateway (Gemini) |
                                                          +-------------------+
```

## Создаваемые файлы

### 1. Edge Function: `supabase/functions/ai-assistant/index.ts`
- Принимает массив сообщений от клиента
- Добавляет системный промпт с описанием всех функций приложения subday (подписки, кофейни, QR-коды, бонусы, стрики, профиль, subFlow лента)
- Вызывает Lovable AI Gateway (`google/gemini-3-flash-preview`) со стримингом
- Обрабатывает ошибки 429 (rate limit) и 402 (нет средств)
- Возвращает SSE-стрим клиенту

### 2. Компонент: `src/components/chat/AiAssistantChat.tsx`
- Sheet (панель снизу), открываемый по кнопке
- Список сообщений (user/assistant) со скроллом
- Поле ввода + кнопка отправки
- Стриминг ответов с обновлением последнего сообщения ассистента токен за токеном
- Обработка ошибок с toast-уведомлениями

### 3. Компонент: `src/components/chat/AiAssistantButton.tsx`
- Плавающая кнопка (FAB) в левом нижнем углу
- Позиционирование над BottomNav
- Анимация появления

### 4. Интеграция в `AppLayout.tsx`
- Добавление кнопки и чата в layout, чтобы ассистент был доступен на всех страницах

### 5. Конфигурация: `supabase/config.toml`
- Добавление записи для функции `ai-assistant` с `verify_jwt = false`

## Системный промпт ассистента

AI будет знать:
- **Главная**: баланс кофе, ближайшие кофейни, рекламные баннеры
- **Подписки**: типы подписок (кофе, напитки), оформление, цены
- **Кофейни**: список, часы работы, расстояние, галерея
- **Профиль**: аватар, имя, уведомления, тема, техподдержка
- **QR-код**: как получить кофе (показать QR бариста)
- **Бонусы и стрики**: система лояльности
- **subFlow**: лента новостей от кофеен
