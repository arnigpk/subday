# Memory: admin/rbac-barista
Updated: now

Интерфейс кабинета партнера адаптируется под роли. Пользователям с ролью 'barista' доступны разделы «Сканер» и «Моя смена» (персональная история списаний, маршрут /partner/my-shift). Бариста видят только те операции, которые они совершили лично (фильтрация по 'scanned_by' = auth.uid()). История включает фильтры по датам (сегодня/неделя/месяц/произвольно) и адресам филиалов. Дашборд, общая история и аналитика партнера для бариста скрыты. По умолчанию показывается фильтр «Сегодня».
