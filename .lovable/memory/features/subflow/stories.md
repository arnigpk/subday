# Memory: features/subflow/stories
Updated: now

Instagram-style Stories in #subFlow with 24-hour expiration. Supports both **images** and **video** (up to 20 seconds; duration validated client-side before upload). The `stories` table has a `media_type` column (`image`/`video`). StoriesBar uses Supabase Realtime for instant updates. **Sorting**: users are ordered by their most recent story timestamp (newest first), with the current user always pinned first — matching Instagram's behavior. When tapping a user's stories, the viewer opens from the **latest (newest) story** so recent content is shown immediately. Creation flow: fullscreen experience with Camera, Gallery, Video, and Record options. Image stories support 8 CSS filters and draggable text overlay rendered via Canvas. Video stories play inline with audio. Features: auto-progress (10s for images, video duration for video), automatic transition between users, instant preloading of adjacent content, and swipe-down to close. Engagement: likes trigger bell notifications, and owners can view a list of users who have seen their story. Viewed status tracked: gradient rings for new content, grey for seen.
