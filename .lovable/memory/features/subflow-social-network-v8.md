Updated: just now

#subFlow is a fully public real-time social network for sharing coffee experiences with a Gen Z/Alpha aesthetic. All users can see authors (names, avatars) of posts, stories, and comments. Posting, story creation, and commenting require an active subscription; non-subscribers see a call-to-action banner: "Купите подписку чтобы публиковать посты, сториз и комментарии в #subFlow". All users can react (💚, 👍, 🔥, 🚀, ⚡️), with a limit of 2 reactions per post. Posts support text and up to 5 photos in a responsive carousel, and authors can edit content within a 60-minute window. The feed uses Supabase Realtime for instant updates and includes infinite scroll. Shop filtering is disabled.
