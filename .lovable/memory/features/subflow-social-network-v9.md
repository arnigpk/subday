# Memory: features/subflow-social-network-v9
Updated: just now

#subFlow — это публичная социальная сеть для обмена впечатлениями о кофейнях в реальном времени, выполненная в эстетике Gen Z/Alpha.
- Видимость: Все пользователи (включая тех, у кого нет подписки) могут видеть ленту, посты, комментарии и личности авторов. Для идентификации используется «Псевдоним #subFlow», который имеет приоритет над реальным именем.
- Ограничения: Публикация постов и написание комментариев доступны только пользователям с активной подпиской. Пользователи без подписки видят баннер: «Купите подписку чтобы публиковать посты и комментарии в #subFlow».
- Реакции: Все пользователи могут ставить реакции в порядке: 💚, 👍, 🔥, 🚀, ⚡️. Лимит — не более 2 реакций на один пост от одного пользователя.
- Функционал: Посты поддерживают текст и до 5 фотографий в карусели. Авторы могут редактировать контент в течение 60 минут после публикации.
- **Сториз полностью удалены** из функционала #subFlow.
- Производительность: Используется Supabase Realtime для мгновенных обновлений и бесконечная прокрутка. Фильтрация по кофейням и функции «Поделиться» отключены.
